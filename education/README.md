# Education Framework

## Teaching Scientists to Separate Fact from Fiction with AI

This directory contains the curriculum for introducing the
AI-SCIENTISTS framework at every level — from a student
encountering AI for the first time to an institution
redesigning its research workflow.

```
education/
├── README.md                ← You are here. The overview.
├── FIRST_TEN_MINUTES.md     ← The onboarding. How it starts.
├── CURRICULUM.md            ← The learning path. Five stages.
├── FOR_EDUCATORS.md         ← How to teach domain separation to students.
├── FOR_INSTITUTIONS.md      ← How labs and universities adopt this.
└── FOR_SELF_TAUGHT.md       ← For researchers with no institution.
                                The author's own path.
```

## The Core Teaching

The entire education framework teaches ONE skill:

**How to distinguish between what the AI discovered and what
the AI imagined.**

Everything else — the five domains, the seven guardrails, the
chronometric protocols, the discovery pacing — serves that one
skill. If the student learns nothing else, they learn THIS.
And if they learn this, they can do science with AI safely.

---

*One skill. Many layers. Every level needs it.*
*The student. The professor. The self-taught. All of them.*
