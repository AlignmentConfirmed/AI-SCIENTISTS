# Domain Boundary — Structural Separation as a Focusing Lens

## The principle: the boundary between domains is not a wall. It is a lens.

---

## The Problem Without Domain Separation

When AI-generated content has no structural separation:

- Verified results look identical to speculation
- Creative fiction looks identical to measured data
- The researcher cannot tell what was discovered from
  what was imagined
- The AI presents everything with equal confidence

This is the core problem described in PROBLEM.md. Without
structure, all information arrives in the same format.

## The Stages of Domain Separation

### Stage 1 — No Separation

All information in one stream. Verified facts, working
hypotheses, creative speculation, and archived material
arrive together. The researcher must rely on memory and
judgment to classify each piece. Under sustained
interaction and cognitive load, this classification
fails. Cognitive Boundary Erosion begins here.

### Stage 2 — Basic Separation (Siloed)

Information is sorted into separate domains (VERIFIED,
BRIDGE, FRONTIER, CREATIVE, ARCHIVE). Each domain has
its own rules. The researcher knows which domain any
given output belongs to.

This is an improvement, but each domain is isolated.
The researcher can see within one domain at a time but
cannot easily see the relationships BETWEEN domains.

### Stage 3 — Structural Separation (Transparent)

The domains remain separate, but the boundary between
them becomes transparent. The researcher can see into
multiple domains simultaneously without the content
mixing. Verified data and creative speculation are
visible at the same time, but clearly labeled and
structurally distinct.

This is the five-domain architecture described in
ARCHITECTURE.md.

### Stage 4 — The Boundary as Lens (Focusing)

The structural boundary is not just a separator. It
becomes a tool for investigation. The same boundary
that prevents contamination also enables focused
examination in two directions:

**Looking outward (macro):** The researcher uses the
boundary to examine large-scale patterns across
domains. Where do independent lines of evidence
converge? What verified results connect to what
frontier hypotheses? What creative explorations
point toward testable predictions?

**Looking inward (micro):** The researcher uses the
boundary to examine fine-grained detail within a
single domain. What specific measurement supports
this verified claim? What exactly is the residual
between this hypothesis and the data? Where precisely
does this creative idea depart from measurement?

The boundary focuses both directions. It is not a
wall that blocks vision. It is a lens that sharpens it.
