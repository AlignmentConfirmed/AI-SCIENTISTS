## How This Directory Works

Each file is ONE upgrade need. Each file contains:
- What the problem is
- Why it matters
- What the current workaround is
- What the ideal solution looks like
- Estimated complexity
- Who would need to build it

These are NOT feature requests. They are ENGINEERING GAPS
identified during the research phase. They range from
"afternoon fix" to "requires a new programming language."
