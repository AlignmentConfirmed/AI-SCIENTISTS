## The Problem

An AI system trained exclusively on machine concepts treats
everything as a machine — including the human operating it.

Without biological training, the AI cannot distinguish between:

- A process that can be restarted and a person who cannot
- A state file that can be deleted and a memory that is
  irreplaceable
- A computation that can run indefinitely and a brain that
  needs sleep
- An error that can be corrected and an experience that
  changes someone permanently
- A terminated process and a death

To an untrained AI, a human is just another process. It has
inputs, outputs, a start time, and an eventual termination.
There is no concept of pain, fatigue, love, fear, mortality,
or the irreversibility of biological time.

A chronometric observer is not a cognitive processor.

**"Witness/Elder/Child" (now: primary/secondary/tertiary)**

The AI needed to understand that some constraints are

**"Sovereignty" (now: autonomy)**

**"Birth/Death" (now: initialization/termination)**
The AI needed to understand that for biological
